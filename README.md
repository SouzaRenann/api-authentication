A secure authentication API built with Java, Spring Boot, Spring Security, JPA, and PostgreSQL.  
The project uses JWT (JSON Web Token) authentication to protect routes and manage user access securely.

###  Features

- User registration and login
- JWT token authentication
- Role-based authorization (Admin/User)
- Protected API routes
- Password encryption with Spring Security
- PostgreSQL database integration
- RESTful API architecture

###  Access Control

The API implements a permission hierarchy between administrators and regular users, allowing better control over protected actions and system management.
